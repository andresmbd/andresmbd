# Ice Cream Order

## Description
This program is a simple ice cream order calculator.
It asks the user to enter the type of product and the quantity they want to buy. The program checks the product selected and assigns a specific price depending on the option: cone, cup, or banana split. Then, it multiplies the price by the quantity to calculate the total cost. Finally, it prints the total amount the user needs to pay.

## How It Works
1. First, the program asks the user to enter the type of product they want to buy. The available options are cone, cup, or banana split.
2. Next, the program asks the user to enter the quantity of the product. This value is converted into an integer so the program can perform calculations.
3. Then, the program checks which product the user selected. It uses conditional statements to assign the correct price for each product. For example, if the user selects cone, the price is 3000. If the user selects cup, the price is 4000. If the user selects banana split, the price is 9000.
4. After determining the price, the program multiplies the price by the quantity to calculate the total cost.
5. Finally, the program prints the total amount that the user needs to pay.
